# DragnCards
Multiplayer online card game written in Elixir, Phoenix, React, and Typescript.

## How to build

- `asdf install erlang 22.3.4.20`
- `asdf local erlang 22.3.4.20`
- `asdf install elixir 1.9.4-otp-22`
- `asdf local elixir 1.9.4-otp-22`

- `mix deps.get`


- `cd backend`
- `mix deps.get`
